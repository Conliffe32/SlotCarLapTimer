# SlotCarLapTimer
This code is for the 1/64 scale HO slot car lap time I built for my AFX race track.  It is to be used with a Raspberry Pi 3.0
The entire project includes 
1) KiCAD files that show the schematic for the electrical aspect of the project.
2) Link to TinkerCAD drawing for 3D printed structures for the project. https://www.tinkercad.com/things/8wsH0qh3mmu-slot-car-lap-timer-v2.
3) And some Visio files that show hot all the modules get connected.

I may also include some video and phots of the project once complete.
